## 使用 electron-icon-builder 可以一次性生成所有端对应的图标

github：https://github.com/safu9/electron-icon-builder

### 安装

```bash
npm install -g  electron-icon-builder
```

### 命令

```bash
electron-icon-builder --input=./build/icon.png --output=./build/icon --flatten
```

--input 为输入文件夹

--output 为输入文件夹

还可以在项目中安装使用
