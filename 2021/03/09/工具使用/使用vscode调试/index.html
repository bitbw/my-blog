使用 vscode 直接调试 js 代码

步骤

  • 将当前标签页设置为需要调试的 js
  • 将需要打断点的地方标上小红点
  • 左侧工具栏点击小虫子（Run and Debug）
  • 点击运行和调试
  • 选择 nodejs 作为调试工具
  • 开始调试

如果只调试 js 代码的话，选择 nodejs 即可

调试界面

总结

使用 vscode 调试非常方便，比直接在浏览器中调式体验更好，可以边调试边改代码，很爽的体验；
