var 变量提升

var 申明的变量会被提升到当前作用域的最顶端
所谓作用域是 function 这样的函数才算是, if else for 等带 {} 的是没有作用域的无论嵌套多少层都会提升

Array(3) 和 Array.from({length:3})

Array(3) 是创建了一个 length: 3 没有经过初始化的数组，数组每一项其实是empty也就是未经初始化
Array.from({length:3}) 是创建了一个 length: 3 经过初始化的数组,数组每一项是 undefind

使用时的区别 Array.prototype.map filter forEach 等， 只有数组中被初始化过元素才会被触发，也就是说未经初始化的元素压根不会被遍历到
