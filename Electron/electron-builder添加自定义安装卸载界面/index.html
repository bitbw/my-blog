注意事项

如果 include 像我一样添加的是卸载页面，会报错：warning 6020: Uninstaller script code found but WriteUninstaller never used - no uninstaller will be

虽然只是警告 但是 编译时默认使用/WX (将警告视为错误 warningsAsErrors为true（默认） )，导致打包中断。
