## nginx

安装

``` bash
sudo apt-get install nginx
```

查看nginx是否安装成功

```bash
nginx -v
```

启动nginx

```bash
service nginx start
```

关闭

```bash
service nginx stop
```

详细教程：https://blog.csdn.net/qq_23832313/article/details/83578836
