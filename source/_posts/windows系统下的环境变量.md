windows 系统下 如需要在命令行中执行某个程序 如：

```bash
typora xx.md
```

需要配置环境变量

找到用户环境变量配置 PATH 将 typora 的可执行文件的所在目录的路径添加进去 ,例如：

```bash
C:\app\Typora\bin
```

注意：只需要添加 exe 文件的目录即可 C:\app\Typora\bin，不需要要加上 typora.exe；

添加完后需要重启命令行工具 ，命令才可以正常执行
