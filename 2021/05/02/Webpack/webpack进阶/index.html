Plugins

Plugin 插件可以用于执行范围更广的任务。包括：打包优化，资源管理，注入环境变量。

原理

Plugin 实例 插件是一个具有 apply 方法的 JavaScript 对象。apply 方法会被 webpack compiler 调用，并且在 整个 编译生命周期都可以访问 compiler 对象。
